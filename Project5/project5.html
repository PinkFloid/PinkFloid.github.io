Training Loss:

We observe that nearly all generated outputs look the same and resemble a blurry mixture of multiple digits. This happens because the input is pure random Gaussian noise, which provides almost no information about which specific MNIST digit the model should reconstruct. With an MSE loss, the model is rewarded for producing an “average” output that minimizes squared error over many possible training images. As a result, instead of generating diverse digits, the network tends to converge to a single prototype that looks like a superposition of many digits, so different random inputs lead to very similar outputs.
